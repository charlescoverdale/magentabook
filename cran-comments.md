## Submission summary

This is a new submission of magentabook 0.1.0.

`magentabook` provides UK HM Treasury Magenta Book policy-evaluation
primitives in R: theory of change, evaluation planning, power and
sample-size calculation (including cluster and stepped-wedge designs),
Maryland Scientific Methods Scale ratings, structured confidence ratings,
light-weight difference-in-differences and interrupted-time-series
estimators, and cost-effectiveness analysis (cost per outcome,
incremental cost-effectiveness ratio, acceptability curves, incremental
net benefit, QALYs and DALYs).

Pure computation: no network calls, no API keys, no external services.
